> 现在我们希望你书写程序，将给定迷宫的 道路网格​，渲染为字符串输出。

## 设计思想 Main Design

- Pythonic: Python代码风格，遵循PEP8

- 静态工厂模式:方便日后扩展，支持不同的输入格式创建迷宫对象。

- 责任链模式：自定义异常类，方便日后扩展更多异常种类。

- 更多设计等待您的探索。
